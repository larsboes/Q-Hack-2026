# Workshop Slides

## Quick Start
- `npm install` — install dependencies
- `npm run dev` — start dev server at localhost:3030
- `npm run build` — build static site to dist/
- `npm run export` — export PDF to docs/handout.pdf (requires playwright-chromium)
- `npm run validate` — validate slides.md

## Dependencies
- @slidev/cli
- @slidev/theme-default
- playwright-chromium (for PDF export)
