## Challenge Format Reminders

- **24 hours** to build + prepare pitch
- Scored on: **Innovation (5)** + **Business Impact & Scalability (3)** + **Technical Implementation (5)** + **UX/Design (3)** + **Pitch (4)** = 20 points total
- Must produce a **functioning MVP / prototype**
- Should be **open-ended** — no predetermined solution

### 1. 🚆 Smart Mobility & Multimodal Transport

> *Directly maps to Exxeta's biggest industry vertical (Deutsche Bahn, Mercedes-Benz)*

**Problem A: AI-Powered Disruption Recovery for Public Transport**
- Trains/buses get delayed → passengers are stranded with no good real-time re-routing across operators
- Build an AI agent that ingests real-time delay data and suggests optimal multimodal alternatives (bus, e-scooter, ride-share, bike) in real-time
- *Why it's good:* Novel (cross-operator), scalable, clear business case, feasible MVP with public GTFS data

**Problem B: Predictive Maintenance Dashboard for Fleet Operators**
- Vehicles (trains, buses, trucks) generate sensor data → predict failures before they happen
- Dashboard visualizing risk scores, maintenance windows, cost savings
- *Why it's good:* Directly relevant to Exxeta's mobility + AI capabilities, strong business impact

**Problem C: Carbon Footprint Tracker for Commuters**
- App that tracks commute patterns and shows CO₂ impact, gamifies switching to greener modes
- *Why it's good:* B-Corp angle, sustainability, easy MVP, engaging UX

## Considerations for Challenge Formulation

If **you** are creating the challenge for Exxeta (i.e., writing the challenge document following the structure guidelines):

1. **Leave room for creativity** — frame it as a problem domain, not a specific solution
2. **Provide mock data / APIs** — teams should spend time building, not sourcing data
3. **Align with Exxeta's strengths** — AI, agentic systems, digital sovereignty, cross-industry
4. **Make it relatable** — students should care about the problem personally
5. **Visual demo potential** — judges need to see something impressive in the pitch
